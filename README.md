# PFE: Retail Performance Optimization - Marjane

## 📝 Project Description
This project was developed as part of my **Project of Final year (PFE) internship at Marjane**. The primary objective is to leverage transactional data to transform customer insights into actionable growth strategies for the retail sector.

By utilizing **Data Science** methodologies, this project provides solutions for:
* **Customer Segmentation:** Implementing an RFM (Recency, Frequency, Monetary) model and clustering to identify high-value VIP clients.
* **Merchandising Optimization:** Conducting Market Basket Analysis (MBA) using Apriori/FP-Growth algorithms to identify product associations for cross-selling.
* **Strategic Decision Support:** Creating an interactive dashboard to monitor key performance indicators (KPIs) and track sales trends.

## 🚀 Key Features
- [ ] **RFM Segmentation:** Classification of customers into distinct segments (VIP, Regular, Occasional, Inactive).
- [ ] **Market Basket Analysis:** Extraction of association rules to identify frequent product bundles.
- [ ] **Data Visualization:** Interactive dashboard development for KPI monitoring.
- [ ] **Business Recommendations:** Tailored strategies for each customer segment to maximize basket size and retention.

## 🛠 Tech Stack
* **Language:** Python
* **Data Science/ML:** Pandas, Scikit-learn, Mlxtend, NumPy
* **Visualization:** Power BI, Matplotlib, Seaborn
* **Tools:** Git, Jupyter Notebooks
